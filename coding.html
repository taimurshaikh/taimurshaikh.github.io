I've been coding for about 4 years now. In that time, I've experimented with lots of different areas. Right now, my projects mostly involve Artificial Intelligence and/or nature-inspired algorithms. Below are some of the coding projects I'm most proud of! PS: source code for all these projects and more can be found at github.com/taimurshaikh

Darwin Runner (Dec. 2020-Present)

For my A-Level CS coursework, I'm developing a visualization of an AI system 'evolving' to play an endless runner game. The system uses a technique called neuro-evolution, which combines the two concepts of neural networks and genetic algorithms. The former is a learning model that uses simplified analogies of the human brain, and the latter is a Reinforcement Learning algorithm that mimicks natural selection and evolution. Coded in C# using the Unity game engine.

Source code.

Music Genetic Algorithm (Mar. 2021)

This is a genetic algorithm that composes music. I'm using a modified version of this algorithm for my EPQ independent research project to generate melodies for an album of lofi music called 'Compile'. You can listen to it in the Music section.

Source code.

Allama-Bot (Dec. 2020 - Mar. 2021)

'Allama-Bot' is a Natural Language Processing (NLP) system that can generate Urdu poetry in the style of Allama Iqbal, a renowned Urdu and Farsi poet. After training the model, I compiled an anthology of 50 poems and published them alongside Land of the Pure, a Dubai-based NGO centred on conversations pertaining to Pakistan. I also wrote an article for their blog about the project
